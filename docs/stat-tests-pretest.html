28  Pre-Tests oder Vortest

Probleme mit dem Pre-Tests ist, dass die Fallzahl oft viel zu klein ist um eine eindeutige Aussage zu treffen.

28.2 Pre-Test auf Varianzhomogenität

\[ \begin{align*} H_0: &\; s^2_A = s^2_B\\ H_A: &\; s^2_A \ne s^2_B\\ \end{align*} \]

28.3 Varianzen sind homogen, Fallzahl hoch

set.seed(202209013)

high_homo_tbl <- tibble(A = rnorm(100, 10, 5),
                        B = rnorm(100, 10, 5)) %>% 
  gather(trt, rsp) %>% 
  mutate(trt = as_factor(trt))
ggplot(high_homo_tbl, aes(trt, rsp, fill = trt)) +
  theme_bw() +
  geom_boxplot() +
  scale_fill_okabeito() +
  theme(legend.position = "none")

leveneTest(rsp ~ trt, data = high_homo_tbl) %>% tidy %>% select(p.value)
# A tibble: 1 × 1
  p.value
    <dbl>
1   0.835
bartlett.test(rsp ~ trt, data = high_homo_tbl) %>% tidy %>% select(p.value)
# A tibble: 1 × 1
  p.value
    <dbl>
1   0.611
lm(rsp ~ trt, data = high_homo_tbl) %>% check_model(check = "homogeneity")

lm(rsp ~ trt, data = high_homo_tbl) %>% check_homogeneity()
OK: There is not clear evidence for different variances across groups (Bartlett Test, p = 0.611).

28.7 Pre-Test auf Normalverteilung

\[ \begin{align*} H_0: &\; \mbox{y ist gleich normalverteilt}\\ H_A: &\; \mbox{y ist nicht gleich normalverteilt}\\ \end{align*} \]

28.8 Approximativ normalverteilt, niedrige Fallzahl

set.seed(202209013)
high_normal_tbl <- tibble(A = rt(10, df = 30),
                          B = rt(10, df = 30)) %>% 
  gather(trt, rsp) %>% 
  mutate(trt = as_factor(trt))
ggplot(high_normal_tbl, aes(trt, rsp, fill = trt)) +
  theme_bw() +
  geom_boxplot() +
  scale_fill_okabeito() +
  theme(legend.position = "none")

ggplot(high_normal_tbl, aes(rsp, fill = trt)) +
  theme_bw() +
  geom_density(alpha = 0.5) +
  scale_fill_okabeito() +
  theme(legend.position = "none")

high_normal_tbl %>% 
  filter(trt == "A") %>% 
  pull(rsp) %>% 
  shapiro.test()

    Shapiro-Wilk normality test

data:  .
W = 0.945699, p-value = 0.61797
high_normal_tbl %>% 
  filter(trt == "B") %>% 
  pull(rsp) %>% 
  shapiro.test()

    Shapiro-Wilk normality test

data:  .
W = 0.89291, p-value = 0.18282
